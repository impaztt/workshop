# 워크샵 게임 웹 서비스

워크샵 현장에서 대형 화면/프로젝터로 진행하는 게임 플랫폼.
진행자가 메인 화면에서 게임을 선택 → 설명 → 본게임을 진행하고,
관리자페이지에서 게임과 콘텐츠를 등록·수정·삭제한다.

## 기술 스택

- **Next.js 15 (App Router) + TypeScript**
- **Tailwind CSS v4** + **Framer Motion** (글래스모피즘 / 애니메이션)
- **JSON 파일 데이터 스토어** (`data/db.json`, 최초 실행 시 12종 게임 자동 시드)
- 파일 업로드: `public/uploads`
- 인증: 단순 비밀번호 + 서명 쿠키

## 실행

```bash
npm install
npm run dev      # http://localhost:3000 (개발)

npm run build && npm run start   # 프로덕션 (행사 당일 권장)
```

## 환경 변수 (`.env.local`)

```
ADMIN_PASSWORD=workshop2026          # 관리자 페이지 비밀번호 (행사 전 변경)
SESSION_SECRET=<긴 랜덤 문자열>       # 쿠키 서명용
```

## 화면

| 경로 | 설명 |
|---|---|
| `/` | 메인 게임 목록 (4×3 카드) |
| `/game/[id]` | 게임 설명 화면 |
| `/game/[id]/play` | 본게임 진행 화면 |
| `/admin` | 관리자 (비밀번호 로그인) |
| `/admin/games` | 게임 목록 관리 (순서·노출·CRUD) |
| `/admin/games/[id]` | 게임별 콘텐츠 관리 |
| `/admin/gifts` | 선물증정 — 선물/참가자 등록 |
| `/admin/settings` | 행사명·타이틀 설정 |

## 구현 범위 (1차)

플레이 가능한 게임 4종: **밸런스 게임 · 이미지퀴즈 · 음악퀴즈 · 선물증정**.
공통 타이머(시작/정지/초기화), 진행 번호(n/전체), 정답·오답 연출, 게임 종료 컨페티 포함.

나머지 8종(이어그리기·줄줄이 말해요·흑백요리사·같이 말해·숨바꼭질·도전골든벨·진실게임·기상미션)은
메인 카드와 설명 화면까지 제공되며, 본게임 화면은 2차 개발 대상이다.
관리자페이지에서는 12종 모두 게임/콘텐츠 정보를 미리 등록할 수 있다.

## 데이터 초기화

관리자 대시보드 하단 "진행 상태 초기화" → 기본 12종 시드 상태로 복원 (등록 콘텐츠 삭제됨).
또는 `data/db.json` 파일을 삭제하면 다음 실행 시 재시드된다.
